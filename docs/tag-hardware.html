The acceleration of public–key cryptanalysis methods using Xilinx Zynq-7000 ★ Posted on November 05, 2018
The Xilinx Zynq-7000 SoC provides new possibilities for increase of efficiency of cryptanalysis methods for public–key systems such as RSA or Diffie–Hellman key exchange algorithm. These cryptosystems are based on the discrete logarithm and integer factorization problem. After brief introduction to numerical methods for solving of these problems there is an introduction to the distributed system that aims to solving of these problems.
